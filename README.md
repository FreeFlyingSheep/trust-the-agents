# Trust the Agents｜信任智能体

Trust the Agents is a short terminal satire game.

《Trust the Agents》是一款短篇终端讽刺游戏。

You manage a small multi-agent system under KPI pressure, trying to survive three rounds of trade-offs between speed, budget, and stability.

你需要在 KPI 压力下管理一个小型多智能体系统，在速度、预算与稳定性之间熬过三轮取舍。
