# Trust the Agents｜信任智能体

Trust the Agents is a short terminal satire game.

《Trust the Agents》是一款短篇终端讽刺游戏。

You manage a small multi-agent system under KPI pressure, trying to survive three rounds of trade-offs between speed, budget, and stability.

你需要在 KPI 压力下管理一个小型多智能体系统，在速度、预算与稳定性之间熬过三轮取舍。

## Notes｜说明

This project is pure vibe coding, just for fun.

本项目属于纯氛围编程，纯属娱乐。

## Simulation｜模拟

You can run the balance simulator with Godot in headless mode:

你可以用 Godot 的 headless 模式运行平衡性模拟：

```bash
godot --headless -s tools/simulate.gd
```

It prints the ending distribution summary directly in the terminal.

它会直接在终端输出结局分布汇总。
